Ch01 - Advanced Selectors

Child and Sibling Selectors

  • Child Combinator using a > sign (e.g. nav > ul) gets the first ul inside nav
  • Adjacent Sibling Combinator using a + sign (e.g. h2 + p) gets the first p after h2
  • General Sibling Combinator using a ~ sign (e.g. h2 ~ p) gets all p's after h2

Attribute Selectors

  • Selecting based on the existence of an HTML Attribute using the attribute name, e.g. a[title]
  • Selecting based on the exact value of an HTML Attribute using an equal sign, e.g. a[type="submit"]
  • Selecting based on the partial contents of an HTML Attribute value using an asterisk and equal sign, e.g. a[href*="twitter"]
  • Selecting based on the beginning of an HTML Attribute value using a caret and equal sign, e.g. a[href^="http"]
  • Selecting based on the end of an HTML Attribute value using a dollar and equal sign, e.g. a[href$=".nz"]
