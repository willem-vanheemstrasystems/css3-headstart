Advanced Selectors

Selecting based on the beginning of an HTML attribute value

The selector targets all the a elements with an href value starting with http, simply by using a caret (^) symbol.

a[href^="http"] { padding-left: 30px; padding-top: 5px; padding-bottom: 5px; background: url(img/external_link_font_awesome_30px.png) left no-repeat; }
