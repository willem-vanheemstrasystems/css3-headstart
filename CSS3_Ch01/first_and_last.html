First and Last

In this example, the pseudo-class looks for the last child in the ul, which is, of course, the last navigation list item, and it removes the border-bottom styling.

ul { border: 1px solid #000; list-style-type: none; }
ul li { border-bottom: 1px solid #000; }
ul li:last-child { border-bottom: none; }

In this example, the pseudo-class looks for the first child in the ul, which is, of course, the first navigation list item, and it sets the background color to red.

ul { border: 1px solid #000; list-style-type: none; }
ul li { border-bottom: 1px solid #000; }
ul li:first-child { background-color: red; }
