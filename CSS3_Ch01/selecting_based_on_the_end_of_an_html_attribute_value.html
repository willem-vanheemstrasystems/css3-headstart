Advanced Selectors

Selecting based on the end of an HTML attribute value

The selector targets all the a elements with an href value ending with .nz, simply by using a dollar ($) sign.

a[href$=".nz"] { padding-left: 30px; padding-top: 5px; padding-bottom: 5px; background: url(img/new_zealand_flag.jpg) left no-repeat; }
